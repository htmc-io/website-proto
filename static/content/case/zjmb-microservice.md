## 案例简介
为某省级公司构建下一代网络管理运维平台。

平台采用 Docker + Kubernetes 的微服务方案。

平台对外提供开放 API 业务网关，内部多个管理系统能同时部署在该平台，并相互隔离。

平台根据不同时间的业务负载量，能快速动态扩缩容。
